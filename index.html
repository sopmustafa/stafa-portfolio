FPV Drone

What it is

A custom-built First Person View (FPV) drone capable of high-speed, low-altitude flight with live video feed to goggles, built entirely from scratch using individual components.

Problem it solves

Provides an affordable, locally-built drone solution for aerial inspection and monitoring — technology usually imported at high cost in Cambodia.

Tools Used

Flight controller · ESCs · Brushless motors · LiPo battery · FPV camera · Python scripts for telemetry

My Role

Solo builder — designed the frame layout, assembled all electronics, tuned PID settings, and programmed the flight logic.

Challenge Overcome

Resolving motor signal interference and achieving stable hovering with correct PID calibration — required multiple test flights and parameter adjustments.

Result / Impact

Successfully built a fully flyable FPV drone with stable performance, demonstrating that Cambodian students can build advanced tech from the ground up.

Robotics C++ · Arduino · Electronics

Remote Control Car

What it is

A custom remote-controlled car built from scratch with a wireless controller, motorized chassis, and programmable speed/direction controls.

Problem it solves

Acts as a foundation for ground-based robotic exploration — learning remote sensing and control fundamentals applicable to real autonomous vehicles.

Tools Used

Arduino · DC motors · L298N motor driver · RF/Bluetooth module · C++

My Role

Full project owner — wired the circuit, wrote the firmware in C++, and assembled the chassis with custom mounting brackets.

Challenge Overcome

Fixing signal lag between controller and car, and preventing brownout resets under high motor load by adding capacitors to stabilize power delivery.

Result / Impact

Delivered a smooth, responsive RC car. Became a live demo at school to inspire younger students in robotics and electronics.

Auto Water Planting System

What it is

An automated plant watering system that reads soil moisture levels and activates a water pump when plants need hydration — no human input required.

Problem it solves

Addresses over/under watering of plants and reduces water waste — a practical solution for households and small farms in Cambodia where water management is important.

Tools Used

Arduino · Soil moisture sensor · Water pump · Relay module · Python

My Role

Designed and built the full system — sensor wiring, pump relay logic, and wrote the Python automation script with threshold-based triggers.

Challenge Overcome

Calibrating moisture sensor readings for different soil types and preventing electrical interference between the pump relay and sensor readings.

Result / Impact

Fully functional system that automatically maintains optimal soil moisture, reducing manual watering effort by 100% and demonstrating practical IoT use in agriculture.
